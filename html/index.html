This is a public, unlimited, ad-free, direct Telegram MTProxy server. The primary purpose of this proxy server is to circumvent connection issues and censorship. It is safe to use and runs 9seconds/mtg server implementation written in Go. A connection is also possible by copy-pasting the code below. Please verify that your Telegram app is updated.

This is a FakeTLS proxy

FakeTLS is a special type of server. It cloaks server responses as https traffic. The certificate also matches the host so that everything is valid from a firewall point of view.

This mode envelops Telegram stream in TLS so it looks (in theory) the same as any TLS1.3 traffic from DPI point of view.

mtg project page README.md 

In other words, this means that the traffic between the Telegram app and this proxy server is hard to distinguish between all other typical internet traffic. There's a lower chance when using this proxy of a firewall block or disconnection.

This also means you from time to time should visit this page to get the updated server secret, so that the current secret and the cloaked response match the proxy active domain.
